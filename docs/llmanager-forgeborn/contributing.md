# 🛠️ Contributing Guidelines

## Local Setup

```bash
git clone https://github.com/HGS-RD/forgeborn-core
npm install
```

## Branching

- `main`: Production
- `feature/*`: New features
- `hotfix/*`: Bug fixes

## Checklist

- [ ] Add docs for any new agent/skill
- [ ] Pass `check:rules`
- [ ] Include test trace logs
